# CourseWorkHypermarket
First grade course work. OOP.

## Build project
1. ### clone project
```
$ git clone https://github.com/ashnaider/CourseWorkHypermarket.git
```
1. ### build project
```
$ cd CourseWorkHypermarket
$ mkdir build && cd build
$ cmake ..
$ make all
```
1. ### run project
```./main```
